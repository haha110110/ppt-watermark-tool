# PPT & PDF 批量加水印保护工具 (Windows 专版)

这是一款专为 Windows 平台设计的图形化工具，旨在批量为 PowerPoint 演示文稿（PPT/PPTX）和 PDF 文件添加水印，并实现不可编辑的**扁平化保护**。

## 🌟 核心功能

*   **多格式支持**：支持直接导入 `.ppt`、`.pptx` 以及 `.pdf`（完美兼容横版格式）。
*   **格式交叉输出**：您可以自由选择将最终文件导出为 **PPTX** 或 **PDF**，支持 PPT 转 PDF 或 PDF 转 PPT。
*   **绝对安全扁平化**：无论输出为哪种格式，每一页都已被强制转换为合并了水印的单张背景图片，彻底杜绝内容被复制、修改或提取文字。
*   **自定义高清水印**：
    *   支持透明背景的 PNG 水印。
    *   **自动适配尺寸**：智能检测原文档是 16:9 还是 4:3 比例。
    *   **灵活缩放**：支持通过滑块或输入精确的小数点百分比来调整水印占屏幕的缩放比例（默认 15%）。
*   **体积与质量控制**：
    *   导出分辨率支持 **1080p** 极致高清与 **720p** 便携体积切换。
    *   支持通过 **30%~100% 的质量滑块**精细调节生成的背景图片压缩率，解决传统转图导致的体积暴增问题。
*   **自动化记忆**：工具会自动记录您上一次使用过的源文件夹、水印路径、输出文件夹以及输出格式配置，下次打开时自动填充，省时省力。
*   **防覆盖机制**：如果选择与源文件相同的输出文件夹，系统会自动在文件名后追加 `_wm` 后缀，保护原件安全。

## 🛠 环境要求与依赖

*   **操作系统**：仅限 **Windows** 系统。
*   **核心依赖**：系统中必须已安装 **Microsoft Office (PowerPoint)** 客户端，否则无法调用 PPT 渲染底层组件。
*   **Python 环境**：Python 3.x (推荐 3.10+)。

## 🚀 安装步骤

1.  克隆或下载本仓库到本地。
2.  在项目根目录打开终端 (Terminal 或 CMD)。
3.  安装依赖库（推荐使用国内镜像源以防网络超时）：
    ```bash
    pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
    ```

> **依赖项说明**：
> *   `pywin32`：用于调用系统的 PowerPoint 组件。
> *   `Pillow`：强大的图片处理与水印拼合引擎。
> *   `PyMuPDF (fitz)`：地球上最快的 PDF 解析与合成引擎。

## 🎯 使用方法

1.  运行主程序：
    ```bash
    python main_gui.py
    ```
2.  在弹出的界面中，依次完成以下配置：
    *   **添加文件**：选择一个或多个需要处理的 PPT/PDF 文件。
    *   **选择水印**：选择您的水印图片（建议使用带透明通道的 `.png`）。
    *   **调节比例**：设置水印相对于页面高度的大小。
    *   **设置质量**：选择输出 1080p 或 720p，并拖动滑块决定 JPG 图片质量（质量越低体积越小）。
    *   **选择输出格式**：选择生成 PPTX 还是 PDF。
    *   **输出文件夹**：选择保存路径。
3.  点击 **“开始处理”**。
4.  程序将在后台静默调用组件进行处理（主界面不会卡顿，且会显示进度提示）。完成后可直接前往输出文件夹查阅文件。

## ⚠️ 注意事项

*   **不可逆操作**：导出的文件不支持重新提取文本或编辑内部元素，请妥善保管好原始源文件。
*   **Office 弹窗**：处理极个别加密或受损文件时，Office 可能会弹出确认修复窗口，遇到卡进度的情况请注意检查任务栏是否有闪烁的 PowerPoint 图标。
